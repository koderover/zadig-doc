## Create a New Environment

> Suitable for K8s YAML projects.

### Test Environment

**Request**

```
POST /openapi/environments?projectKey=<project identifier>
```

**Body Parameter description**

| Parameter Name             | Description         | Required                                | Type |
| ------------------ | ------------ | ----------------------------------- | ---- |
| `env_key`          | Environmental Identification     | string                              | Yes |
| `cluster_id`       | K8s cluster ID  | string                              | Yes |
| `namespace`        | K8s namespace | string                              | Yes |
| `registry_id`      | image registry ID  | string                              | Yes |
| `env_configs`      | Environment Configuration     | [][EnvConfig](#EnvConfig)           | No   |
| `global_variables` | Global Variables     | [][GlobalVariable](#GlobalVariable) | No   |
| `services`         | Service List     | [][Service](#Service)               | No   |

<h4 id="!">EnvConfig Parameter description</h4>

| Parameter Name                | Description                                                                          | Required   | Type |
| --------------------- | ----------------------------------------------------------------------------- | ------ | ---- |
| `yaml_data`           | YAML content for environment configuration                                                          | string | Yes |
| `common_env_cfg_type` | The type of environment configuration, optional values are as follows:<br> `ConfigMap`<br> `Ingress`<br> `Secret`<br> `PVC` | string | Yes |

<h4 id="!">GlobalVariable Parameter description</h4>

| Parameter Name  | Description     | Required   | Type |
| ------- | -------- | ------ | ---- |
| `key`   | The key of the variable | string | Yes |
| `value` | Value of variable | string | Yes |

<h4 id="!">Service Parameter description</h4>

| Parameter Name         | Description                        | Required                        | Type |
| -------------- | --------------------------- | --------------------------- | ---- |
| `service_name` | Service name                    | string                      | Yes |
| `type`         | Service type, just specify it as `k8s` | string                      | Yes |
| `containers`   | Mirror information of the service              | [][Container](#Container)   | Yes |
| `variable_kvs` | Service variable configuration                | [][VariableKV](#VariableKV) | No   |

<h4 id="!">Container Parameter description</h4>

| Parameter Name       | Description                                                              | Required   | Type |
| ------------ | ----------------------------------------------------------------- | ------ | ---- |
| `image`      | The complete mirror address, for example: `koderover.demo.com/public/service1:latest` | string | Yes |
| `image_name` | The image name                                                          | string | Yes |
| `name`       | Service Component Name                                                      | string | Yes |

<h4 id="!">VariableKV Parameter description</h4>

| Parameter Name                | Description             | Required   | Type |
| --------------------- | ---------------- | ------ | ---- |
| `key`                 | The key of the variable         | string | Yes |
| `value`               | Value of variable         | string | Yes |
| `use_global_variable` | Whether to use global variables | bool   | No   |
