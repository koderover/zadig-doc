This article introduces integrating gateway components in Zadig.

## Integrating Gateway in Zadig

In Zadig, navigate to `System Settings` -> `Integration` -> `Gateway` to add gateway information, as shown in the following figure.

![gateway](../../../_images/gateway_1.png)

Parameter Description:

- `System Identifier`: System identifier used to identify the system
- `Access Address`: Gateway System access address
- `Admin Key`: Admin Key used for authentication
