Zadig v4.2.1 was officially released on March 24, 2026.

## Feature List

**Environment and Service**
- Record version when modifying service replicas in K8s YAML projects
- Fine-grained permissions for environment restart and rollback

**Deployment and Release**
- Deploy/rollback tasks support merge and overlay strategies

## Version Upgrade Process

::: warning
If MongoDB is deployed in Replica Set mode, please confirm that system databases such as admin/local/config/test are not used as Zadig's business database. If they are used, please manually migrate the data first.

If the current system version < v4.2.0, please upgrade to v4.2.0 first. For specific upgrade steps, please refer to [v4.2.0 Upgrade Method](/en/Zadig%20v4.2/release-notes/v4.2.0/#version-upgrade-process), and then upgrade to v4.2.1 according to the following method.
:::

Please execute the corresponding upgrade process according to different installation methods:

- [Script Installation](/en/Zadig%20v4.2/install/helm-deploy/#upgrade)
- [Helm Installation](/en/Zadig%20v4.2/install/helm-deploy/#upgrade)
- [Official Installation](/en/Zadig%20v4.2/stable/install/#upgrade)
