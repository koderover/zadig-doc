## 2026 Roadmap
### AI 能力
- [x] AI 发布风险检测 - 智能识别发布过程中的潜在风险
- [x] AI 环境巡检 - 自动化检测集群健康状态
- [x] AI 效能诊断 - 一键分析流水线效率瓶颈
### 工作流引擎
- [x] 共享变量体系，系统与项目变量复用，简化配置
- [x] 服务重启任务，快速恢复业务，减少停机时间
- [x] APISIX 变更任务，网关配置变更，灵活管控
- [x] Apollo 配置搜索，快速定位配置，提升调试效率
- [x] Lark 应用通知，实时推送状态，协作更高效
- [x] 敏感变量支持，全局变量可加密，保护密钥安全
- [x] 主机环境固定值，部署任务环境参数可固化
- [x] 自定义任务动态变量，按需生成，适配复杂场景
- [x] 部署日志增强，发布过程透明，问题可追溯
- [x] 飞书工作项追溯，关联工作项，信息无缝流转
- [x] 主机构建使用模板，构建配置复用，效率提升
- [x] 用例通过率配置，可设置阈值，灵活适配不同质量标准
- [ ] 禁用缓存执行，灵活控制，避免残留干扰
- [ ] MR 事件过滤，忽略描述修改，减少无效触发
- [ ] 审批时间变量格式化，详情可读，操作更直观
- [ ] DMS 工单串行执行，工单有序推进，降低变更风险
- [ ] 执行条件获取任务变量和状态，灵活智能
- [ ] WebHook Payload 获取，触发参数可提取，使用更灵活
- [ ] 工作流通知使用变量，通知人员动态定制，触达更精准
### 发布计划
- [x] 执行人指定，责任到人，协同更清晰
- [x] 失败重试，提升发布成功率
- [x] 权限细化拆分，精细管控，降低越权风险
- [ ] 计划复制，复用成熟流程，提升发布效率
- [ ] 日历视图呈现，发布时间直观可控
- [ ] 拖拽排序，灵活调整节奏，适配业务节奏
- [ ] 外部检测 Hook，增加检测说明，增强体验
### 服务部署及模板管理
- [x] 副本数版本记录，变更历史可查，快速溯源
- [x] 部署策略选择，合并与覆盖按需配置，灵活适配业务
- [x] Ingress 路径展示，服务访问透明化
- [x] Helm manifest 差异，发布前对比，降低误发布风险
- [ ] CRD 资源支持，K8s YAML 部署覆盖更多资源类型
- [ ] 部署队列机制，避免并发覆盖问题，保障发布稳定
- [ ] 变更窗口期控制，业务低峰发布，减少影响范围
- [ ] YAML 模板从代码库同步，版本可追溯
### 安全与权限
- [x] 环境权限细化，重启、回滚、调整副本数独立权限控制
- [ ] API 调用安全防护，Token 加密显示、开关控制、支持重置
- [ ] 登录安全增强，验证码防复用、用户名防枚举、登录信息加密
- [ ] 只读角色 system-read-only，保障系统基础安全
- [ ] 支持多因素认证 (MFA)，系统安全升级
OpenAPI & 扩展
- [x] 系统访问地址配置，内网回调支持
- [x] 共享变量管理，项目与系统级别全覆盖
- [x] Helm 场景 OpenAPI 补充，使用场景更全面
- [x] 发布计划全局变量，OpenAPI 传入执行参数
- [x] 子环境管理 OpenAPI，新建与删除能力
- [x] 协作模式管理 OpenAPI，新建与删除能力
- [x] 工作流执行通知 OpenAPI，自定义通知用户
- [x] 多服务任务协同 OpenAPI，测试、代码扫描、通用任务
- [x] 自定义任务变量输出，任务间支持数据传递
- [ ] 自定义任务定制页面，满足个性化诉求
- [ ] 项目插件入口，生态扩展更便捷
