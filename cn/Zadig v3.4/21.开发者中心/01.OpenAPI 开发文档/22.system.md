## 列出系统操作日志

### 请求

```
GET /openapi/system/operation
```

### Query

| 参数名     | 类型   | 描述                                                                                                         | 必填 |
| ---------- | ------ | ------------------------------------------------------------------------------------------------------------ | ---- |
| searchType | string | 搜索类型，可选值：<br>`all`：全部<br>`project`：项目<br>`user`：用户<br>`function`：功能<br>`status`：状态码 | 是   | 是 |
| projectKey | string | 项目标识                                                                                                     | 否   |
| username   | string | 用户名                                                                                                       | 否   |
| function   | string | 功能                                                                                                         | 否   |
| status     | int    | 状态码                                                                                                       | 否   |
| perPage    | int    | 每页数量                                                                                                     | 是   |
| page       | int    | 页码                                                                                                         | 是   |

### 返回
