## 查看容器实时日志

**请求**

```
GET /openapi/logs/sse/pods/<pod名称>/containers/<容器名称>
```

**Query 参数说明**

| 参数名       | 说明          | 类型   | 必填 |
| ------------ | ------------- | ------ | ---- |
| `projectKey` | 项目标识      | string | 是   |
| `envName`    | 环境名称      | string | 是   |
| `tails`      | 日志tails行数 | string | 是   |

**返回示例**

::: details
该接口返回的是一个 SSE 流，返回的数据格式如下：

```
"log message"
```
:::
