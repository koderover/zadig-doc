本文按照字母升序介绍 Zadig 平台中的相关术语。

## PR

指代表变更请求，在不同的代码源中具体表述如下：

| 代码源 | 全称 | 
|-------- | ---- |
| GitHub | Pull request |
| GitLab | Merge request |
| Gerrit | Change |
| Codehub | Merge request |

## Push

将本地的代码变更提交到远端的代码库。
